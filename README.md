# Clerivo

Clerivo est une plateforme "local first" hébergée sur Raspberry Pi, conçue pour assister les agences immobilières suisses dans la gestion locative.

## Structure du projet

- **apps/** : Code source (Frontend React, Backend Node.js)
- **config/** : Configuration infrastructure (Nginx, Systemd)
- **data/** : Données persistantes (Base de données, Documents, Logs) - *À sauvegarder*
- **docs/** : Documentation (CDC, Plans de bataille)
- **scripts/** : Scripts de maintenance (Setup, Backup, Restore)
